# Real Estate AI Assistant

A Retrieval-Augmented Generation (RAG) based AI assistant that answers real estate queries using a custom knowledge base.

## Features

- RAG-based question answering
- Multi-turn conversation memory
- Document retrieval using vector database
- Source citations
- Graceful handling of unavailable information
- Streamlit dashboard
- Login authentication
- Chat history management

## Tech Stack

- Python
- Streamlit
- LangChain
- Google Gemini LLM
- FAISS Vector Database

## Run Application

Install dependencies:

pip install -r requirements.txt

Run:

streamlit run app.py
